WIDS

Waterborne Intruder Detection System

 

Naval Coastal Systems Center

Panama City, FL

GTRI

Atlanta, GA

 

1979-1982

 

Griff Jay

gcon@peak.org

WIDS was a major GTRI research project to adapt radar technology for tracking surface vessels near shorelines and through ship channels. WIDS was a multi-processor system which integrated real-time signal processing and pattern detection on custom microprocessors with output to early vector graphics displays. Three bodies of software - microprocessor assembler, structured Fortran app, and high-level graphics - communicated in real-time.

For all the leading-edge technologies that were part of this project, the most significant innovation was the programming standards that we evolved in order to manage the multi-level complexity of the software and hardware. Between phases #2 and #3 of the project, we streamlined, standardized, and documented the code to a level where by phases #3 and #4 we observed orders of magnitude improvement in productivity. Changes could literally be made in an afternoon that had taken programmers two to four weeks back in phases #1 and #2. This is the project where I learned firsthand what makes the difference between hacker code and professional software.
