At United Airlines, the APMS Reports module is in operational use in producing the airline's quarterly FOQA safety report. I developed, installed, and trained the staff on this module, and I customized it for the UA Flight/Events database. This software supplanted a previous spreadsheet application which had proven to be too time-consuming and an irritatingly repetitive exercise. The number of days required to produce the quarterly report was reduced from 15 to approximately 4.

APMS is a NASA research project to statistically process, display, and analyze commercial airline flight data. This VB application takes data from MS Access, implements an advanced graphics control to display the parameter traces and the statistical components, and uses OLE to automatically output charts to PowerPoint and data tables to Excel.
