The Aerial Indexing project was a major ERDAS project for DIA. I was the project director and principal developer of a suite of applications to display georeferenced satellite imagery on which the analyst could interactively locate any of several camera aperture footprints. Calculating map projections in real-time, the footprint on the image reflected the useful analytical area on the reconnaissance photos.

All the geography, camera, and photo information was stored in a dBase structure that fed a comprehensive database which could be accessed by the President's national security advisors. This application ran on nine separate PC workstations, which we linked together via one of the earliest AT&T StarLan peer-to-peer networks. I wrote both the database utility and the StarLan usage functions, plus I also produced end-user documentation and operations instructional manuals.
