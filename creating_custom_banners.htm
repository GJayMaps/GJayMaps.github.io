Creating Custom Banners

This is an example of a "custom banner", an image I created using a digital photo of a quilt combined with some text effects from ImageComposer.  You can use clip art, digital photos, animations, whatever you are authorized to use.

HOW TO:  Simply position your cursor in the first paragraph on the page, and choose Insert, Picture, from File.

Details:  

File Type File size
.jpg ("jay-peg") 22KB

A .jpg image is usually solid, i.e. no transparent background, and very sharp.

This is the same source image saved as a .gif file - can you see any difference?

Pros  - it is smaller and will appear (load) faster on the web page; it has a transparent background

Cons  - it's a little "fuzzier"

Details:  

File Type File size
.gif 18KB

Same basic idea, a little fancier.  You can put color "behind" a banner .gif image using a table. 

HOW TO:  Choose:

Table, Insert, Table, 1 x 1

The table appears as a single cell (probably without borders so you'll see dotted lines instead of solid like this):

 

Select the table, then choose:

Table, Properties, Cell, Background, & choose a color.

 

Position your cursor inside the table, then choose Insert, Picture, from File.

The cell expands automatically to fit the image.  You can drag the edges to enlarge the cell even more.

You can even put an animated GIF into your banner!

  P.S. Always put at least a HOME button at the bottom of each page!
